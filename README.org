
* Installation

1. Install nix
2. Install nix-darwin
3. Install home-manager:

   #+begin_src sh
     nix-channel --add https://github.com/nix-community/home-manager/archive/master.tar.gz home-manager
     nix-channel --update
   #+end_src

4. Clone this repo into ~~/.nixpkgs~
5. Run:

   #+begin_src sh
     darwin-rebuild switch
   #+end_src
